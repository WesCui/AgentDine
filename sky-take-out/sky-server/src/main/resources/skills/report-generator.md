# 报告生成模板

## 经营报告格式

### 周报模板
```
## 灵食 AgentDine经营周报 ({start_date} ~ {end_date})

### 📊 关键指标
| 指标 | 本周 | 上周 | 环比 |
|------|------|------|------|
| 营业额 | ¥X | ¥Y | ±Z% |
| 订单数 | X单 | Y单 | ±Z% |
| 客单价 | ¥X | ¥Y | ±Z% |
| 订单完成率 | X% | Y% | ±Z% |
| 新增用户 | X | Y | ±Z% |

### 🏆 销量Top5
1. 菜品A — X单
2. 菜品B — Y单
...

### 💡 AI 经营建议
1. ...
2. ...
3. ...
```

### 月报模板
包含周报所有内容，额外增加：
- 月度趋势图描述
- 分类销量占比
- 用户复购率
- 下月经营预测

### 生成规则
- 营业额数据保留2位小数
- 百分比保留1位小数
- 建议不超过5条，每条不超过50字
- 优先给出可执行的 actionable 建议
