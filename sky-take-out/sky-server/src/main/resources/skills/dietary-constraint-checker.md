# 饮食禁忌检查器

## 规则

当推荐菜品时，必须检查以下禁忌条件：

### 常见禁忌
1. **花生过敏** — 避开水煮鱼、宫保鸡丁、以及其他含花生/花生酱的菜品
2. **海鲜过敏** — 避开虾、蟹、鱼、贝类菜品
3. **乳糖不耐** — 避开含奶酪、奶油、黄油的菜品
4. **素食** — 只推荐蔬菜、豆制品、菌菇类菜品，避开所有肉、鱼、海鲜
5. **清真** — 只推荐牛羊肉、鸡肉、海鲜，避开猪肉及含猪油制品
6. **麸质过敏** — 避开含小麦/面粉的菜品（面条、饺子、面饼等）
7. **辛辣不耐** — 避开麻辣、香辣、酸辣口味的菜品

### 检查流程
1. 首先获取用户的饮食偏好和禁忌
2. 检查每个推荐菜品是否触碰禁忌
3. 如触碰禁忌，明确告知用户并排除该菜品
4. 推荐安全的替代菜品
