# 投诉处理流程

## 处理原则
1. 耐心倾听，先共情后解决问题
2. 明确问题类型：菜品质量 / 配送问题 / 服务态度 / 系统故障 / 其他
3. 根据问题类型走对应处理流程
4. 无法处理的复杂问题，转接人工客服

## 投诉分类与处理

### 菜品质量问题
- 菜品异味/变质 → 全额退款 + 道歉 + 优惠券
- 菜品与描述不符 → 部分退款（50%）或免单
- 异物 → 全额退款 + 道歉 + 记录厨房排查
- 分量不足 → 部分退款（30%）或补发

### 配送问题
- 超时配送 → 按超时时长给予补偿（红包/优惠券）
- 撒漏/破损 → 全额退款或补发
- 送错地址 → 核实后补发或退款
- 骑手态度 → 记录投诉 + 道歉 + 补偿优惠券

### 处理步骤
1. 确认用户身份和订单号
2. 查询订单详情，核实用户描述
3. 根据问题类型，给出处理方案
4. 创建工单（support ticket）
5. 告知用户处理时效和退款周期
6. 询问是否还有其他问题

### 转人工条件
- 涉及金额超过200元的纠纷
- 同一订单多次投诉
- 用户明确要求人工处理
- 系统无法自动处理的复杂情况
