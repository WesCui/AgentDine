# 后厨工作流

## 订单处理优先级
1. VIP用户订单 → 优先处理
2. 催单订单 → 优先处理
3. 超时风险订单（距预计完成 <5分钟）→ 优先处理
4. 普通订单 → 按时间顺序

## 出餐时间标准
- 凉菜/冷饮：3-5分钟
- 热菜（炒制）：5-10分钟
- 热菜（炖煮）：10-20分钟
- 套餐（含3+菜品）：15-25分钟

## 库存预警
- 单品日销量 > 日均销量的2倍 → 库存紧张警告
- 单品连续3日零库存 → 建议下架或补货
- 套餐所需菜品缺货 → 预警并建议临时下架套餐
