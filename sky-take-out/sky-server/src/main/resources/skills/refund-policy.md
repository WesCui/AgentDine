# 退款政策

## 退款条件矩阵

| 场景 | 退款比例 | 处理时效 | 备注 |
|------|---------|---------|------|
| 待付款取消 | 自动取消 | 即时 | 无需商家介入 |
| 待接单取消 | 100% | 1-3工作日 | 商家确认后原路退回 |
| 已接单取消（商家同意） | 50-100% | 1-5工作日 | 视菜品制作进度 |
| 已接单取消（商家拒绝） | 0% | - | 可申诉 |
| 配送中超时30分钟+ | 10-30% | 1-3工作日 | 按超时梯度 |
| 菜品质量问题（有证据） | 80-100% | 1-3工作日 | 需拍照举证 |
| 菜品质量问题（无证据） | 30-50% | 1-5工作日 | 首次信任用户 |
| 撒漏破损 | 100% | 1-3工作日 | 需拍照举证 |
| 少送/漏送 | 按漏送菜品价值 | 1-3工作日 | 核实后处理 |
| 重复扣款 | 退回多扣金额 | 3-5工作日 | 需财务核实 |

## 退款方式
- 原路退回（微信支付 → 微信零钱，银行卡 → 银行卡）
- 退款红包（可用于下次消费，不可提现）
- 优惠券补偿（满减券/折扣券，有效期7-30天）

## 不退款情形
- 因用户自身原因（地址错误、电话不通）导致的配送问题
- 超时完成订单后24小时内未反馈质量问题
- 已知口味偏好但未备注导致的"不好吃"
- 恶意退款申请（系统会标记并限制账号）
